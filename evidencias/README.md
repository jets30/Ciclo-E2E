# Evidencias de Ejecución

Este directorio contiene evidencias generadas por la ejecución de la suite.

- `playwright-report/`: copia del reporte HTML generado por Playwright.

Para generar nuevas evidencias:
1. Ejecutar `npm test`.
2. Revisar el reporte con `npm run test:report`.
3. Copiar el contenido de `playwright-report/` a `evidencias/playwright-report/` para versionar el reporte.

Ejemplo de evidencia disponible:
- `evidencias/playwright-report/index.html`
